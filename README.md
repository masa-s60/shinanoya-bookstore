# 信濃屋書店

横須賀中央駅前にある老舗書店のコーポレートサイトです。

## 制作時期

* 2024年

## 使用技術

* HTML
* CSS
* JavaScript
* PHP
* JSON

## 主な機能

* 最新情報管理機能
* 画像アップロード機能
* Googleマップ埋め込み

最新情報管理機能について

対応機能

* 最新情報の追加
* 最新情報の削除
* 画像アップロード
* 最新順表示

## 画像アップロード機能について

対応形式

* jpg
* jpeg
* png
* gif
* webp

非対応形式

* heic
* heif

非対応理由

* Safari以外の主要ブラウザ（Chrome、Firefox、Edge）がHEIC表示に対応していないため
* 現在のPHP環境ではHEIC画像をJPEGへ変換できないため

## 実装内容

* レスポンシブ対応
* 初回訪問アニメーション
* JSONデータ管理
* パララックス表現、など

## 備考

* 投稿データはJSONで管理
* 投稿は日時単位で最新順表示
* 公開用リポジトリのため、メールアドレス等の機密情報は除外
* 不要なライブラリ・ファイルを整理済み
