# 信濃屋書店

老舗書店のWebサイト

## 制作時期

* 2024年

## 使用技術

* HTML
* CSS
* JavaScript（vivus.js、BookSnap）
* PHP
* JSON

## 実装内容

* 制作ページ：4ページ（メインページ、教科書取扱い店ページ、個人情報保護ページ、最新情報投稿ページ）
* レスポンシブ対応
* PHPによる最新情報管理機能（以下に詳細を記載）
* JSONデータ管理（以下に詳細を記載）
* 初回訪問アニメーション及びその他アニメーション（以下に詳細を記載）
* 各種JS（以下に詳細を記載）

### PHPによる最新情報管理機能について

* 最新情報の追加
* 最新情報の削除
* 画像アップロード
* 最新順表示


【画像アップロード機能について】

対応形式
* jpg
* jpeg
* png
* gif
* webp

非対応形式
* heic
* heif

非対応理由
* Safari以外の主要ブラウザ（Chrome、Firefox、Edge）がHEIC表示に対応していないため
* 現在のPHP環境ではHEIC画像をJPEGへ変換できないため

### JSONデータ管理について
* 投稿作成日時を保持
* 投稿日時を基準に最新順で表示
* トップページおよび投稿ページへ自動反映
* 投稿管理のためアクセスログを記録

* パララックス表現
* スクロール連動アニメーション
* フェードインアニメーション
